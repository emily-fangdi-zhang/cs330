PacMan

Image

Description

PacMan is modeled after PacMan the arcade version with exact dimensions and functons. I utilized Northwestern's GE211 game engine to customize queries allowing ghosts to move with different characteristics and personalities. I designed one continuous level that support audio, 2D graphics and keyboard interaction. All code is written in C++ and follows the model-view-controller design pattern. PacMan features active and inactive ghosts, big and small points, and a PacMan with three lives.

Photo-App

Image

Description

Photo-App, modeled after Instagram, is a full-stack web application that allows users to view and interact with their friend's posts, and see suggested followers. I created the routes that allow a user to comment and like a post, as well as follow other users. I also implemented relational database to store user information with Python Flask and SQLalchemy. At the end, I deployed the app to the Heroku cloud platform in order to allow the user to connect to the server and make API calls.
